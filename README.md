# Weather Dashboard

A simple web application that allows users to search for a city and view the current weather and a 7-day forecast using the Open-Meteo API.

## Technologies Used

- HTML
- CSS
- JavaScript
- Open-Meteo APIs (Geocoding & Forecast)

## How to Run the Project

### Option 1 (Quickest)

1. Download or clone the repository
2. Open `index.html` in your browser

### Option 2 (Recommended)

Run a local server to avoid browser restrictions:

- **VS Code**: Install “Live Server” → right-click `index.html`
- **Python**:

```bash
python -m http.server

```
